# AI Career Copilot 튜토리얼

## 서비스 소개
AI Career Copilot은 취업 준비생을 위한 AI 기반 커리어 진단 서비스입니다.
GitHub 포트폴리오, 이력서, 채용공고를 분석하여 취업 경쟁력을 높이는 실질적인 피드백을 제공합니다.

---

## 튜토리얼 1: 종합 커리어 진단 (처음 사용자 추천)

### 목표
GitHub URL, 이력서, 채용공고를 한 번에 입력하여 전체적인 커리어 현황을 진단받습니다.

**1단계: GitHub URL 입력** - 본인의 GitHub 프로필 URL을 입력합니다. 예시: https://github.com/사용자명
**2단계: 이력서 업로드 또는 붙여넣기** - PDF 파일 업로드 또는 텍스트를 직접 붙여넣을 수 있습니다. 지원 파일 형식: PDF, TXT, MD
**3단계: 채용공고 URL 입력** - 지원하려는 채용공고 URL을 1~5개 입력합니다. 현재 잡코리아(jobkorea.co.kr) 공고를 지원합니다.
**4단계: 분석 결과 확인** - 기술스택 일치도, 깃 커밋 활동, 레포 기술 커버리지, AI 총평을 확인합니다.

---

## 튜토리얼 2: GitHub 포트폴리오 분석

### 목표
GitHub 레포지토리들이 채용공고의 요구 스킬과 얼마나 맞는지 확인합니다.

**1단계:** GitHub 레포 URL을 최대 5개 입력합니다. 예시: https://github.com/사용자명/레포명
**2단계:** 채용공고 URL을 최대 5개 입력합니다.
**3단계:** 결과 확인 - 포트폴리오 등급, 전체 적합도(%), 강점 기술, 보완 기술, 레포별 상세, 채용공고별 매칭 상세

---

## 튜토리얼 3: 스킬 갭 분석

### 목표
현재 보유 기술과 채용공고 요구 기술의 차이를 파악하고 보완 방법을 추천받습니다.

**1단계:** GitHub 레포 URL 입력 (선택) - 실제 코드 기반 기술이 분석에 포함됩니다.
**2단계:** 이력서 텍스트 입력 (선택) - 이력서 기반 기술도 함께 분석됩니다.
**3단계:** 채용공고 URL 입력 (필수)
**4단계:** 결과 확인 - 입증된 기술, 부족한 기술, 우선순위 기술, 기업 적합도 순위, 추천 프로젝트

---

## 튜토리얼 4: 이력서-GitHub 연계 분석

### 목표
이력서에 적힌 기술이 실제 GitHub 코드에서 증명되는지 확인합니다.

**1단계:** 이력서 텍스트를 붙여넣거나 파일을 업로드합니다.
**2단계:** GitHub 사용자명만 입력합니다 (URL 전체 불필요). 예시: kimcoding
**3단계:** 주요 기술스택을 콤마로 구분하여 입력합니다. 예시: React, TypeScript, Node.js
**4단계:** 결과 확인 - 검증 완료 기술, 미검증 기술, 새로 발견된 기술, AI 종합 평가, 보완 조언

---

## 튜토리얼 5: AI 면접 질문 생성

### 목표
자기소개서와 채용공고를 바탕으로 예상 면접 질문과 답변 팁을 받습니다.

**1단계:** 작성한 자기소개서를 텍스트로 붙여넣습니다.
**2단계:** 채용공고 URL 또는 텍스트를 입력합니다 (선택). 잡코리아 URL은 자동으로 공고 내용을 가져옵니다.
**3단계:** 결과 확인 - 기술 구현, 트러블슈팅, 시스템 설계, 협업소통, CS 기초, DevOps 카테고리별 질문, 출제 의도, 핵심 키워드, 답변 팁, 예시 답변

---

## 튜토리얼 6: 자기소개서 비교 분석

### 목표
원본 자기소개서와 수정된 자기소개서를 비교하여 개선 포인트를 파악합니다.

**1단계:** 처음 작성한 자기소개서를 붙여넣습니다.
**2단계:** 수정 또는 AI로 개선한 버전을 붙여넣습니다.
**3단계:** 결과 확인 - 전체 요약, 개선된 표현 목록, 추가된 경험, 강화된 기술, 아직 부족한 부분

---

## 자주 묻는 질문 (FAQ)

**Q: GitHub 레포가 private이면 분석이 되나요?**
A: 현재는 public 레포지토리만 분석 가능합니다.

**Q: 이력서를 반드시 PDF로 올려야 하나요?**
A: 아닙니다. 텍스트를 직접 복사해서 붙여넣을 수 있습니다.

**Q: 채용공고는 어느 사이트 URL이 지원되나요?**
A: 현재 잡코리아(jobkorea.co.kr) URL을 지원합니다. 다른 사이트 공고는 텍스트를 직접 붙여넣어 사용하세요.

**Q: 채용공고를 몇 개까지 입력할 수 있나요?**
A: 최대 5개까지 입력 가능합니다.

**Q: 분석 결과는 저장되나요?**
A: 분석 기록은 히스토리 메뉴에서 확인할 수 있습니다.

**Q: 이력서 없이도 스킬 갭 분석이 되나요?**
A: 네, 이력서와 GitHub URL 모두 선택 입력입니다. 단, 둘 다 입력할수록 분석이 정확해집니다.
