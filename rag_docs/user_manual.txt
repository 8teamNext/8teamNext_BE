# AI Career Copilot 사용 설명서

## 서비스 개요
AI Career Copilot은 취업 준비생이 GitHub 포트폴리오, 이력서, 채용공고를 연결하여 커리어 경쟁력을 높일 수 있도록 돕는 AI 분석 서비스입니다.
주요 기능: 포트폴리오 분석 / 스킬 갭 파악 / 면접 준비 / 자기소개서 개선

---

## 기능 1: 종합 커리어 진단

### 설명
GitHub, 이력서, 채용공고를 한 번에 입력하여 기술스택 일치도, 깃 활동량, 레포 커버리지를 종합 진단하는 핵심 기능입니다.

필요 입력값: GitHub URL(필수), 이력서 텍스트(필수), 채용공고 URL(필수, 최대 5개, 잡코리아 지원)

출력 결과: 포트폴리오 등급, 전체 매칭 비율, 기술스택 일치도(%), 깃 커밋 활동, 레포 기술 커버리지, GitHub 분석 상세, 이력서 분석 상세, 스킬 갭과 학습 로드맵, 추천 프로젝트, AI 총평

활용 팁: 여러 회사의 채용공고를 한 번에 넣으면 어느 포지션에 가장 적합한지 비교할 수 있습니다.

---

## 기능 2: GitHub 포트폴리오 분석

### 설명
GitHub 레포지토리와 채용공고를 비교하여 포트폴리오의 강점과 보완점을 분석합니다.

필요 입력값: GitHub 유저이름, 채용공고 URL(필수, 최대 5개)

출력 결과: 포트폴리오 등급, 전체 채용 적합도(%), 강점 기술, 보완 기술, README 개선 제안, 레포 상세(언어, README 품질, 스타 수, 커밋 수), 채용공고별 매칭 상세(일치 기술, 부족 기술, 매칭 비율)

활용 팁: README 품질이 미흡인 레포가 있다면 프로젝트 설명, 기술스택, 실행 방법을 추가하세요.

---

## 기능 3: 스킬 갭 분석

### 설명
현재 보유 기술(GitHub + 이력서)과 채용공고 요구 기술을 비교하여 우선적으로 배워야 할 기술과 추천 프로젝트를 제시합니다.

필요 입력값: GitHub 유저이름, 이력서 텍스트, 채용공고 URL(최대 5개)

출력 결과: 입증된 기술, 부족한 기술, 발견된 기술, 우선순위 기술, 기업 적합도 순위, 추천 프로젝트(제목, 설명, 기술, 난이도, 아키텍처)

---

## 기능 4: 이력서-GitHub 연계 분석

### 설명
이력서에 기재한 기술이 실제 GitHub 코드에서 증명되는지 검증합니다.

필요 입력값: 이력서 텍스트(필수), GitHub 사용자명(필수, URL 전체 아닌 사용자명만, 예: kimcoding), 주요 기술스택(필수, 콤마 구분, 예: React, TypeScript, Node.js)

출력 결과: 전체 평가, 이력서 기재 기술 목록, GitHub 확인 기술 목록, 검증 완료 기술, 미검증 기술, 새로 발견된 기술, 보완 조언, 업데이트 제안

활용 팁: 미검증 기술이 많다면 해당 기술을 활용한 토이 프로젝트를 GitHub에 올리는 것을 권장합니다.

---

## 기능 5: AI 면접 질문 생성

### 설명
자기소개서와 채용공고를 바탕으로 실제 면접에서 나올 법한 예상 질문과 답변 가이드를 생성합니다.

필요 입력값: 자기소개서(필수, 텍스트 붙여넣기), 채용공고(선택, 잡코리아 URL 또는 텍스트)

출력 결과: 각 질문별 카테고리, 질문, 출제 의도, 핵심 키워드, 답변 팁, 예시 답변
질문 카테고리: 기술 구현 / 트러블슈팅 / 시스템 설계 / 협업소통 / CS 기초 / DevOps
채용공고 입력 시 추가: 공고 요약, 요구 스킬, 자소서 일치 역량, 자소서에 없는 역량

활용 팁: 잡코리아 URL을 입력하면 공고를 자동 크롤링하여 더 맞춤화된 질문을 생성합니다.


---

## 마이페이지 기능

### 프로필 설정
이름, GitHub 사용자명, 기본 이력서, 기본 자기소개서를 등록할 수 있습니다.
프로필에 이력서와 자소서를 등록하면 각 분석 기능에서 별도 입력 없이 자동으로 적용됩니다.

### 분석 히스토리
실행한 분석의 날짜, 종류, 요약 결과를 기록하며 불필요한 히스토리는 삭제 가능합니다.

---

## 입력 제한 및 주의사항

채용공고 URL은 최대 5개, 잡코리아(jobkorea.co.kr)만 지원. 다른 사이트는 텍스트 직접 붙여넣기.
이력서 파일은 PDF(스캔 이미지 PDF 불가), TXT, MD 지원. 텍스트 붙여넣기도 가능.
GitHub 레포는 public 레포지토리만 분석 가능, 최대 5개.
채용공고 내용은 법적 이슈로 저장되지 않으며 URL만 보관됩니다.

---

## 에러 안내

GitHub 사용자를 찾을 수 없습니다: GitHub 사용자명 재확인 필요
job_urls는 최대 5개까지 가능합니다: 채용공고 URL을 5개 이하로 줄이기
PDF에서 텍스트를 추출할 수 없습니다: 스캔 이미지 PDF 대신 텍스트 PDF 사용 또는 직접 붙여넣기
잡코리아 채용공고를 불러오지 못했습니다: URL 확인 또는 공고 텍스트 직접 입력
요청 body가 없습니다: 필수 입력 항목 확인 필요
